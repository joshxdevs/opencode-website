Features

Built for developers
who want to move fast.

Every detail is designed to get out of your way and let the AI do the work.

Provider Abstraction

Plug in any AI provider — Google, OpenAI, Anthropic and more. Switch models without changing your workflow.

Local Secure Storage

API credentials live in ~/.opencode/ with strict 0600 permissions. Your keys never leave your machine.

Model Discovery

Connects to models.dev to fetch, filter, and cache remote model schemas, context limits, and tool-call capabilities.

Agentic Loop

Out-of-the-box tool-calling framework with list_files, read_file, and run_command. The LLM autonomously inspects and acts on your workspace.

Architecture

Three layers.
One seamless loop.

A clean separation of concerns that keeps the codebase modular and extensible.

01

CLI Routing commands/

Parses arguments and flags via Commander.js. The entry point for every user interaction — clean, declarative, and extensible.

02

Core Library lib/

Handles all business logic — secure credential management, network caching, model discovery, and the agent execution loop.

03

Presentation ui/

Abstracts terminal styling and standardizes all stdout/stderr formatting. Keeps output consistent across every command.

Installation

Up and running
in 60 seconds.

1

Install globally

Requires Node.js 18+ installed on your system.

npm install -g @joshxdevs/opencode
2

Add your API key

Securely store your provider credentials. Keys are saved locally with strict file permissions.

opencode providers login --provider google --api_key YOUR_KEY
3

Set your model

Browse available models and set your default for agent execution.

opencode model set google/gemini-2.5-flash
4

Run the agent

Point it at any directory. The agent reads your filesystem, runs commands, and reports back.

opencode agent -p "Analyze this codebase and summarize it."
Usage

Every command,
exactly where you need it.

Providers
# Login a provider
opencode providers login \
  --provider google \
  --api_key YOUR_KEY

# List active providers
opencode providers list

# Remove a provider
opencode providers logout \
  --provider google
Models
# List all available models
opencode models

# Filter by provider
opencode models --provider google

# Force refresh cache
opencode models --refresh

# Set active model
opencode model set google/gemini-2.5-flash
Agent
# Run a prompt
opencode agent -p "List all files"

# Analyze a directory
opencode agent -p \
  "Summarize this codebase"

# Debug a project
opencode agent -p \
  "Find bugs in the src folder"
